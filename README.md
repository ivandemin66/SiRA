# SiRA

## Описание
SiRA (Sentiment Analysis Application) — это контейнерное Java-приложение, предназначенное для анализа тональности текста. Оно использует простую модель ИИ для определения эмоциональной окраски текста (положительная, отрицательная или нейтральная) и предоставляет результаты через REST API.

## Основные возможности
- Анализ тональности текста с использованием простейшей логики.
- REST API для взаимодействия с приложением.
- Контейнеризация с использованием Docker.
- Развертывание в локальном Kubernetes-кластере с помощью Minikube.
- Настройка мониторинга и балансировки нагрузки.

## Стек технологий
- **Язык программирования**: Java
- **Сборка**: Maven
- **Контейнеризация**: Docker
- **Оркестрация**: Kubernetes (Minikube)

## Как запустить

### Локальный запуск
1. Убедитесь, что у вас установлен JDK 17 и Maven.
2. Соберите проект:
   ```bash
   mvn clean package
   ```
3. Запустите приложение:
   ```bash
   java -jar target/*.jar
   ```
4. Приложение будет доступно по адресу: `http://localhost:8080`.

### Запуск в Docker
1. Соберите Docker-образ:
   ```bash
   docker build -t sentiment-app .
   ```
2. Запустите контейнер:
   ```bash
   docker run -p 8080:8080 sentiment-app
   ```

### Развертывание в Minikube
1. Убедитесь, что Minikube установлен и запущен.
2. Примените манифесты Kubernetes:
   ```bash
   kubectl apply -f k8s-deployment.yaml
   ```
3. Получите внешний IP-адрес сервиса:
   ```bash
   minikube service sentiment-app-service
   ```

## Мониторинг и балансировка нагрузки
- Приложение настроено на использование `readinessProbe` и `livenessProbe` для проверки состояния контейнеров.
- Балансировка нагрузки осуществляется через объект Kubernetes `Service` с типом `LoadBalancer`.

## Отчет
В рамках проекта будет подготовлен аналитический отчет о текущих тенденциях в области ИИ и контейнеризации на основе статей с arxiv.org.
